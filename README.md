# Book Is On & On 웹서비스
## aivle0317

## 사용 스택

1. React
2. Node.js (v16.15.1)
3. Yarn

## 실행 방법

1. `npm install -g yarn`: yarn 설치
2. `yarn start`: 로컬 서버 실행
3. `yarn dev`: 로컬 테스트 서버 실행(예정)

## 주의 사항

1. .gitignore에 저장되지 않는 항목이 있습니다.
2. .gitignore 추가시 파일에 있는 링크로 추가하시길 바랍니다.

## 목적

1. 프로젝트 시작시 기본적으로 갖추어야할 js web server를 만들기 위해서 입니다.
